###### 1. Write the `correct answer` from the following options and give an explanation (2-5 lines).

```javascript
let greeting;
greetign = {};
console.log(greetign);
```

- A: `{}`
- B: `ReferenceError: greetign is not defined`
- C: `undefined`

<details><summary><b>Answer</b></summary>
<p>

#### Answer: B

<i>In the let variable there is greeting. But in console log, there is greetign. Which is not defined. So, the answer will be not defined.</i>

</p>

###### 2. Write the `correct answer` from the following options and give an explanation (2-5 lines).

```javascript
function sum(a, b) {
  return a + b;
}

sum(1, "2");
```

- A: `NaN`
- B: `TypeError`
- C: `"12"`
- D: `3`

<details><summary><b>Answer</b></summary>
<p>

#### Answer: C

<i>In this problem, there is one number and one string. There is a plus sign. This code will add 2 beside 1. Because 2 is not a number. We know that in js, we use plus sign when we want to addition in numbers and also for string to add beside one string to another. So, the answer will be 12</i>

</p>
</details>
